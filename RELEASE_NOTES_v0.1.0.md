# Fanqie Clock v0.1.0

First public Windows prototype release.

## Highlights

- Floating Pomodoro timer ball
- macOS-inspired control window
- Full-screen break reminder
- Tray integration and background running
- Launch on startup support
- Local config persistence

## Included in This Release

- Windows packaged build: `FanqieClock-win64.zip`
- Source code
- Project README

## Notes

- This is a prototype release focused on core workflow and UI direction.
- Windows SmartScreen may show a warning because the app is not code-signed.
- Please run the app from the full extracted folder, not from a standalone copied `.exe`.

## Known Limitations

- Windows-only for now
- No installer yet; distribution is zip-based
- Additional UI polish and runtime validation are still in progress

## Feedback Areas

- Floating ball usability
- Main window layout and visual polish
- Break reminder readability
- Tray behavior and startup experience
